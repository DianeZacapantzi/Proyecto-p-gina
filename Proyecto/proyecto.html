¡Bienvenidos!

Esta tienda busca brindarte información sobre el mundo del maquillaje, así como mostrarte algunos productos que podrían ser de tu agrado y que podrías conseguir.

El maquillaje es la práctica de decorar la piel y otras partes visibles el cuerpo para resaltarlas. También se define como maquillaje todo producto usado para decorar o arreglar rasgos faciales y corporales de una persona; este se compone de diversos elementos con funciones determinadas para cada parte.
